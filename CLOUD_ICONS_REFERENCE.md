## 1. React Icons (react-icons)

### Font Awesome Icons (fa)
```tsx
import { FaCloud, FaCloudRain, FaCloudSun, FaCloudMoon, FaCloudShowersHeavy, FaCloudDownload, FaCloudUpload } from 'react-icons/fa'
```

**Available Cloud Icons:**
- `FaCloud` - Basic cloud
- `FaCloudRain` - Cloud with rain
- `FaCloudSun` - Cloud with sun
- `FaCloudMoon` - Cloud with moon
- `FaCloudShowersHeavy` - Heavy rain cloud
- `FaCloudDownload` - Cloud download
- `FaCloudUpload` - Cloud upload

**View all:** [Font Awesome Cloud Icons](https://fontawesome.com/icons?d=gallery&q=cloud)

### Material Design Icons (md)
```tsx
import { MdCloud, MdCloudQueue, MdCloudDone, MdCloudOff, MdCloudUpload, MdCloudDownload, MdCloudCircle } from 'react-icons/md'
```

**Available Cloud Icons:**
- `MdCloud` - Basic cloud
- `MdCloudQueue` - Cloud with queue indicator
- `MdCloudDone` - Cloud with checkmark
- `MdCloudOff` - Cloud disabled
- `MdCloudUpload` - Cloud upload
- `MdCloudDownload` - Cloud download
- `MdCloudCircle` - Cloud in circle

**View all:** [Material Design Cloud Icons](https://fonts.google.com/icons?icon.query=cloud)

### Feather Icons (fi)
```tsx
import { FiCloud, FiCloudRain, FiCloudSnow, FiCloudLightning } from 'react-icons/fi'
```

**Available Cloud Icons:**
- `FiCloud` - Basic cloud
- `FiCloudRain` - Cloud with rain
- `FiCloudSnow` - Cloud with snow
- `FiCloudLightning` - Cloud with lightning

### Heroicons (via react-icons) (hi)
```tsx
import { HiCloud } from 'react-icons/hi'
```

**Available Cloud Icons:**
- `HiCloud` - Basic cloud (outline)

### Bootstrap Icons (bs)
```tsx
import { BsCloud, BsCloudRain, BsCloudSnow, BsCloudLightning, BsCloudDrizzle, BsCloudHail } from 'react-icons/bs'
```

**Available Cloud Icons:**
- `BsCloud` - Basic cloud
- `BsCloudRain` - Cloud with rain
- `BsCloudSnow` - Cloud with snow
- `BsCloudLightning` - Cloud with lightning
- `BsCloudDrizzle` - Cloud with drizzle
- `BsCloudHail` - Cloud with hail

- `CloudIcon` - Basic cloud (outline or solid)

## 3. Phosphor Icons (phosphor-react)

```tsx
import { Cloud, CloudRain, CloudSnow, CloudLightning, CloudFog, CloudDrizzle, CloudHail } from 'phosphor-react'
```

**Available Cloud Icons:**
- `Cloud` - Basic cloud
- `CloudRain` - Cloud with rain
- `CloudSnow` - Cloud with snow
- `CloudLightning` - Cloud with lightning
- `CloudFog` - Cloud with fog
- `CloudDrizzle` - Cloud with drizzle
- `CloudHail` - Cloud with hail

**Weights available:** thin, light, regular, bold, fill, duotone

**View:** [Phosphor Icons - Cloud](https://phosphoricons.com/)

## üìù Usage Example

Once you choose an icon, here's how to use it:

```tsx
// Example: Using Font Awesome Cloud
import { FaCloud } from 'react-icons/fa'

<FaCloud className="h-5 w-5" />

// Example: Using Heroicons Cloud
import { CloudIcon } from '@heroicons/react/24/outline'

<CloudIcon className="h-5 w-5" />

// Example: Using Phosphor Cloud
import { Cloud } from 'phosphor-react'

<Cloud size={20} weight="regular" />
```
